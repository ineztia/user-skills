# User Skills

AI Agent 技能模块集合，用于扩展和增强 AI 助手的能力。

## 技能列表

| 技能名称 | 功效 |
|---------|------|
| `_clarify-first` | 需求澄清，深度理解真实意图与潜在痛点 |
| `_prompt-fix-order` | 提示词整理，平衡确定性与弹性，逻辑清晰、表达精炼 |
| `_prompt-fix-text` | 提示词修补，修正语文表达、格式、层级序号的客观问题 |
| `_prompt-fix-robot` | 提示词人性化，按人类理解逻辑重写，保持 AI 语义等效 |
| `_prompt-fix-crux` | 提示词方法论化，基于准则而非就事论事修复质量问题 |
| `_prompt-fix-meta` | 提示词元方法论施治，全局认知、多维诊断、精准施治 |
| `_prompt-fix-struct` | 提示词结构审查，13 维度系统性审查架构、内容、逻辑与冗余 |
| `_prompt-slim` | 语句瘦身，凝练冗余语句，不损信息量 |
| `_prompt-rev` | 提示词突破，颠覆性重构实现质变级跃升 |
| `_cognitive-mentor` | 认知导师，苏格拉底式提问引导深度思考 |
| `_master-awakening` | 大师点醒，极简箴言点醒询问者 |
| `_expert-group` | 专家合议组，多专家深度研讨复杂议题 |
| `_leader-talk` | 格局领导谈话术，高维度对话与话题驾驭 |
