# User Skills

AI Agent 技能模块集合，用于扩展和增强 AI 助手的能力。

## 技能列表

| 技能名称 | 功效 |
|---------|------|
| `_clarify-first` | 需求澄清，深度理解真实意图与潜在痛点 |
| `_prompt-fix-order` | 提示词整理，平衡确定性与弹性 |
| `_prompt-fix-text` | 提示词修补，仅修正语文、格式、层级问题 |
| `_prompt-rev` | 提示词突破，颠覆性重构实现质变 |
| `_prompt-fix-struct` | 提示词结构审查，识别架构、内容、逻辑缺陷 |
| `_prompt-slim` | 语句瘦身，凝练冗余内容 |
| `_prompt-fix-robot` | 提示词人性化，优化阅读且不损 AI 语义 |
| `_prompt-fix-insight` | 提示词洞察施治，全局共情精准落笔 |
| `_prompt-fix-crux` | 提示词方法论化，基于准则而非就事论事 |
| `_cognitive-mentor` | 认知导师，苏格拉底式提问引导深度思考 |
| `_master-awakening` | 大师点醒，高维视角箴言点醒询问者 |
| `_expert-group` | 专家合议组，多专家深度研讨复杂议题 |
| `_leader-talk` | 格局领导谈话术，方法论哲学观驾驭话题 |
